##トップ画面
- 画面上部navバーがあり、画像,岩手魅力ドットコム、ホームを押すことでホーム画面に遷移する
- それぞれの魅力紹介ページ、ライターページ遷移が可能
- トップ画面では、それぞれの商品のカルーセルを見ることが可能
- 画面中部にはこのWEBサイトの更新情報を見ることが可能
---
